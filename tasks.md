# Флоу пользователя:

1. Заходит на сайт
2. Нажимает "Зарегистрироваться в Timeweb" → переходит по нашей реферальной ссылке
3. Создаёт аккаунт в Timeweb, пополняет баланс (получает x2 на первое пополнение)
4. Копирует свой API-ключ из панели Timeweb
5. Вставляет его на сайте
6. Управляет серверами через чат

# Кто за что платит:

• Юзер → Timeweb — за серверы напрямую через свой аккаунт
• Timeweb → нам — реферальный процент с приведённых клиентов
• LLM — работает за счёт подписки юзера на ChatGPT, не за наш счёт
