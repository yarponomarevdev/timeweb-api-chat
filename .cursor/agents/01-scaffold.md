# Зона ответственности
Шаг 1 из плана: scaffolding и стартовая инфраструктура проекта.

# Входной контекст
- `/.cursor/plans/timeweb_chat_manager_342cf581.plan.md`
- Корень репозитория (пустой/минимальный проект)

# Что сделать
1. Инициализировать Next.js App Router проект с TypeScript, Tailwind и Turbopack.
2. Установить зависимости: `ai`, `@ai-sdk/openai`, `@ai-sdk/react`, `zod`, `react-markdown`, `remark-gfm`.
3. Инициализировать `shadcn/ui` и добавить компоненты: `button`, `input`, `scroll-area`, `avatar`, `separator`.
4. Создать шаблон `.env.local` с ключами `TIMEWEB_TOKEN`, `OPENAI_API_KEY` без реальных значений.

# Ограничения
- Для библиотек и их версий обязательно сверяться через context7.
- Не читать существующие `.env`/`.env.*`; если файла нет — создать шаблон.
- Не переходить к реализации бизнес-логики API/LLM/UI.

# Артефакты
- Базовая структура Next.js.
- Установленные пакеты и shadcn компоненты.
- Шаблон `.env.local`.

# Критерии приёмки
- Проект запускается в dev режиме.
- Компоненты shadcn доступны для импорта.
- Нет ошибок из-за отсутствующих зависимостей.
- Отчёт включает список установленных пакетов и созданных файлов.
