# Дизайн: центрированный первый экран с анимацией

**Дата:** 2026-03-10
**Задача:** DEVTEAM-149
**Статус:** Утверждён

## Цель

После авторизации (ввода API-ключей) показывать центрированный экран с чатом и inline-подсказками. После отправки первого сообщения — анимированный переход чата вниз с эффектом «парения».

## Состояния интерфейса

### Состояние 1: Пустой чат (centered)

Условие: `messages.length === 0`

- Сайдбар скрыт
- Three.js canvas на весь экран: медленно плавающие частицы (`#10a37f`, низкая плотность, opacity 0.4–0.6)
- Центр экрана:
  - Логотип `evolvin.cloud`
  - Подзаголовок: «Управление серверами через AI»
  - Увеличенный textarea (min-height: 80px)
  - Чипы-подсказки под полем (без смайликов):
    - Мои серверы
    - Создать сервер
    - Баланс
    - SSH-ключи
    - Бэкапы
    - Маркетплейс
- Клик по чипу → вставляет текст в поле ввода

### Состояние 2: Переход (отправка первого сообщения)

Последовательность анимации (~600ms):

1. **Чипы** — `AnimatePresence` fade-out + slide-down (150ms)
2. **Частицы Three.js** — рассеиваются вверх и гаснут (400ms)
3. **Поле ввода** — Framer Motion `layout` + spring `{ stiffness: 280, damping: 28 }` едет вниз
4. **Область сообщений** — fade-in сверху (300ms, delay 100ms)
5. **Сайдбар** — slide-in слева (250ms, delay 150ms)
6. **После приземления** — `box-shadow: 0 -8px 32px rgba(16,163,127,0.1)` — эффект «парения»

### Состояние 3: Активный чат

Стандартный макет с сайдбаром. Чипы скрыты.

**Возврат к состоянию 1:** при очистке чата (messages.length становится 0) — обратная анимация.

## Технический стек

| Компонент | Технология |
|-----------|------------|

| Анимация layout | `framer-motion` — `motion.div` + `layout` prop |
| Чипы и переходы | `framer-motion` — `AnimatePresence` |
